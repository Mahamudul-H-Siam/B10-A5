Important FAQ

What is Document Object Model (DOM)?

The Document Object Model (DOM) is a way for web browsers to represent a webpage so that programming languages like JavaScript can interact with it. The DOM represents the HTML document as a tree. Each node in the tree is an object corresponding to a part of the document, such as elements, attributes, and text.

How do you can select an element from the DOM ?

To select an element from the DOM using JavaScript, there are several methods. These methods allow us to find and interact with specific parts of the webpage.

getElementById
This method selects an element by its unique id attribute.
getElementsByClassName
This method selects all elements that share a certain class. It returns a collection
getElementsByTagName
This method selects all elements by their tag name.
querySelector
This method selects the first element that matches a CSS-style selector. You can use id, class, or element tags.
querySelectorAll
This method selects all elements that match a CSS-style selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegationis a technique in the DOM where a single event listener is attached to a parent element to handle events from its child elements. Instead of adding individual event listeners to each child, you add one listener to a common ancestor and take advantage of event bubbling.
Event Delegation is a useful pattern that allows us to write cleaner code, and create fewer event listeners with similar logic.With event delegation, we create fewer event listeners and perform similar events-based logic in one place.
