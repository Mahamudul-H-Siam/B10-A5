Important FAQ

What is Document Object Model (DOM) ?

The Document Object Model (DOM) is a way for web browsers to represent a webpage so that programming languages like JavaScript can interact with it. The DOM represents the HTML document as a tree. Each node in the tree is an object corresponding to a part of the document, such as elements , attributes, and text.

How do you can select an element from the DOM?

To select an element from the DOM using JavaScript, there are several methods. These methods allow us to find and interact with specific parts of the webpage.
Here are the most common ways:
getElementById
This method selects an element by its unique id attribute.
getElementsByClassName
This method selects all elements that share a certain class.
getElementsByTagName
This method selects all elements by their tag name.
querySelector
This method selects the first element that matches a CSS-style selector.
querySelectorAll
This method selects all elements that match a CSS-style selector. It returns a list of all matching elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events from its child elements. Instead of adding individual event listeners to each child, you add one listener to a common ancestor and take advantage of event bubbling.
Event Delegation is a useful pattern that allows us to write cleaner code, and create fewer event listeners with similar logic.With event delegation, we create fewer event listeners and perform similar events-based logic in one place. This makes it easier for us to add and remove elements without having to add new or remove existing event listeners.
